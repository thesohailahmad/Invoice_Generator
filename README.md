# 🧾 Invoice Generator

A Python CLI application to create, manage, and save invoices. Built with Object-Oriented Programming (OOP) concepts.

---

## 📋 Features

- Create invoices with client information and multiple services
- View all invoices (summary list)
- View full invoice details by ID
- Mark invoices as Paid
- Save invoices to JSON file for persistence
- Input validation on all fields
